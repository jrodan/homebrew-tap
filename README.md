# Homebrew Tap (jrodan)

This is a personal Homebrew tap for macOS tools.

## Casks

- **Brightness Control**: A lightweight menu bar app to control the brightness of your internal and external displays using DDC/CI and hybrid software dimming.

## Installation

```bash
# Add the tap
brew tap jrodan/tap

# Install Brightness Control
brew install --cask brightness-control
```

## Update

```bash
brew update
brew upgrade --cask brightness-control
```
